# Northstar Market Backend

This backend serves the data and API routes for a personal full-stack ecommerce app.

## Development

1. Install dependencies with `npm install`.
2. Start the API with `npm run dev`.
3. The server runs on port `3000` unless you override it with environment configuration.

## Notes

- Product images are served from the local `images/` directory.
- Seed data lives in `defaultData/`.
